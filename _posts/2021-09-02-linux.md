## linux user commands 한 줄 요약

인자는 man (command)로 확인
간단한 linux 명령어 정리(한줄요약)

- ls : list
- sort : 정렬
- uniq : 중복제거
- grep : 조건 맞는거 출력
- sed : 필터
- head : 파일 앞부분만 보여줌
- tail : 파일 뒷부분만 보여줌
- tr : 치환
- cat : 연결해서 보여줌
- tac : 거꾸로 출력
- tee : input -> output + fwrite
- less : more보다는 더 많은 기능 있음
- more : 파일 내용 확인, 화면 단위로 출력
- echo : print 같은거임
- touch : 파일 날짜 수정(없으면 생성)
- which : 실행 파일 경로 찾아줌
- export : 쉘 변수를 환경변수로 저장(수출)
- wc : word count
- cut : 파일 각 라인에서 특정부분 제거하거나 추출
