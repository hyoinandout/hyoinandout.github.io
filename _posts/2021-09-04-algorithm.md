백준 연구소 문제를 푸는 도중에 완벽하게 설계하고 특정 값을 넣으면 결과도 잘 나오는데 자꾸 원 배열이 바뀌어서 애를 먹었었다.
배열 복사는 a.copy()를 쓰면 된다고 해서 썼었는데 깊은 복사의 형태가 아닌 얕은 복사의 형태가 되어서(메모리 참조값까지 복사가 되어서) 자꾸 input으로 받은 배열의 값이 변경되어 반복문을 실행할 수 없었다. 나중에 결국 2차원 이상의 배열은 a.copy()가 아닌,

```
import copy
b = copy.deepcopy(a)
```

와 같은 copy 모듈의 deepcopy 함수를 사용해야함을 알게 되었다.
나중에 2차원 이상 배열을 복사할 일이 생길 때 유용하게 쓰자고 이렇게 기록을 남긴다.
